  <action>
Rewrite install.sh preserving the existing structure but adding cross-platform support. The script should:

**1. Parse arguments:**
- `--minimal` flag sets MINIMAL=true (default false)
- Unknown flags print usage and exit

**2. OS detection at top:**
```bash
OS="$(uname -s)"
case "$OS" in
  Linux)  PKG_MGR="pacman" ;;
  Darwin) PKG_MGR="brew" ;;
  *)      echo "Unsupported OS: $OS" >&2; exit 1 ;;
esac
```

**3. Package arrays:**
```bash
CORE_PACMAN=(stow zsh fzf zoxide tmux neovim git)
EXTRA_PACMAN=(bash)

CORE_BREW=(stow zsh fzf zoxide tmux neovim git)
EXTRA_BREW=(bash)
EXTRA_CASKS=(ghostty alacritty)
```

Note: ghostty on Arch is a regular package; on macOS it's a cask. Alacritty is similar. Keep casks separate from formula packages.

**4. Install logic:**
- Linux: `sudo pacman -S --needed ${CORE_PACMAN[@]}` always; `${EXTRA_PACMAN[@]}` if not --minimal; ghostty and alacritty via pacman as regular packages if not --minimal
- Darwin: Auto-install Homebrew if missing using official install script. After installing brew, eval shellenv so brew is available for the rest of the script. `brew install ${CORE_BREW[@]}` always; `${EXTRA_BREW[@]}` if not --minimal; `brew install --cask ${EXTRA_CASKS[@]}` if not --minimal. Use `|| true` or warn-and-continue for cask installs (ghostty cask name may vary).

**5. Zinit install:** Keep as-is (only relevant for zsh, works on both platforms).

**6. Conflict checks:** Keep existing checks AND add new ones:
- `~/.config/shell/config.d/*` files (alias, env, path, secrets.encrypted)
- `~/.config/powershell/profile.ps1`
- `~/.config/alacritty/alacritty.toml`
- `~/.config/alacritty/linux.toml`
- `~/.config/alacritty/macos.toml`

**7. PACKAGES array (stow):**
```bash
PACKAGES=(nvim zsh claude git ssh tmux ghostty bash shell powershell alacritty)
```
All packages stowed on all platforms per user decision. Configs self-filter via platform includes.

**8. Stow command:** Keep `stow --no-folding "$pkg"` (works identically on macOS with brew-installed stow).

**9. Change shell prompt:** Keep as-is.

**10. Post-install notes:** Update secrets decrypt path:
```
1. Decrypt secrets: cd $DOTFILES_DIR/shell/.config/shell/config.d && ansible-vault decrypt secrets.encrypted --output secrets
```

Keep other post-install notes (TPM, nvim, zinit).

**Important:** Keep `set -euo pipefail`. Use `|| true` for non-critical installs (casks) so the script continues if a cask name is wrong.
